# Changelog

All notable changes to the GNSS workspace are recorded here. Crate-local
release notes live beside each package so a reader can separate workspace
changes from package-specific API or behavior changes.

This repository is still in `0.1.0` development. Entries under `Unreleased`
describe work that has not been cut as a published release.

### Release Notes
